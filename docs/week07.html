ノート

CS/DS基礎: データ可視化とリテラシー

データ可視化の原則

データ可視化で最も大切なのは、「何を伝えたいか」を明確にしてから「どう見せるか」を決めることです。目的のない可視化は、ただの飾りになってしまいます。

料理で例えると、「誰に食べてもらうか」（受け手）と「何を味わってほしいか」（伝えたいこと）を決めてからレシピ（見せ方）を考える。高級な食材を並べても、意図がなければ美味しい料理にはならない。

グラフの種類と使い分け

グラフ適している場面例
棒グラフカテゴリ間の量の比較曜日ごとのメッセージ数
折れ線グラフ時間の経過に伴う変化月ごとのアクティブメンバー数の推移
散布図2つの数値の相関関係メッセージ数 vs ファイル数

グラフの嘘に騙されない

グラフの嘘（データリテラシー）

  • Y軸の切り取り: 0から始めない棒グラフは差を誇張する
  • スケール操作: 軸の間隔を変えると印象が変わる
  • チェリーピッキング: 都合の良い期間だけを切り取る
  • 3Dグラフ: 奥行きで面積が歪み、正確な比較ができない

データを見るとき、常に「このグラフは何を隠しているか」を考えるクセをつけましょう。

記述統計の基本

CSVデータを集計するときに使う基本的な統計量です。Pythonでは組み込み関数だけで計算できます。

統計量意味Python
平均（mean）データの中心的な値sum(data) / len(data)
最大値（max）最も大きい値max(data)
最小値（min）最も小さい値min(data)

サンプル1

基本的なCSV読み込み（テキスト表示）

CSVファイルの先頭10行を読み込み、日付とアクティブメンバー数を画面に表示します。

Python ── sample1_basic.py
import py5
import csv

data = []
with open('slack_time.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    count = 0
    for row in reader:
        if count >= 10:
            break
        date = row['日付']
        active = int(row['アクティブなメンバー数（日次）'])
        data.append({'日付': date, 'アクティブ': active})
        count += 1

def setup():
    py5.size(600, 400)
    font = py5.create_font('MS ゴシック', 20)
    py5.text_font(font)

def draw():
    py5.background(255)
    py5.fill(0)
    py5.text_size(18)
    py5.text("CSVデータ表示", 50, 30)
    py5.text_size(14)
    for i, row in enumerate(data):
        y = 60 + i * 25
        py5.text(f"{row['日付']}: {row['アクティブ']}人", 50, y)

py5.run_sketch()

ポイント解説

  • csv.DictReader はCSVの1行目をキーとして辞書のリストを作る
  • int() で文字列を整数に変換（CSVのデータはすべて文字列）
  • enumerate() でインデックスとデータを同時に取得
サンプル2

複数列のデータ表示 + 記述統計

複数の列を読み込んで表形式で表示し、平均・最大・最小を計算します。

Python ── sample2_multi.py
import py5
import csv

data = []
with open('slack_time.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    count = 0
    for row in reader:
        if count >= 8:
            break
        data.append({
            '日付': row['日付'],
            'アクティブ': int(row['アクティブなメンバー数（日次）']),
            'メッセージ': int(row['投稿されたメッセージ数']),
            'ファイル': int(row['アップロードファイル数'])
        })
        count += 1

def setup():
    py5.size(800, 500)
    font = py5.create_font('MS ゴシック', 20)
    py5.text_font(font)

def draw():
    py5.background(255)
    py5.fill(0)
    py5.text_size(20)
    py5.text("複数列データ表示", 50, 30)

    py5.text_size(14)
    py5.text("日付", 50, 70)
    py5.text("アクティブ", 200, 70)
    py5.text("メッセージ", 320, 70)

    for i, row in enumerate(data):
        y = 100 + i * 25
        py5.text(row['日付'][-5:], 50, y)
        py5.text(f"{row['アクティブ']}人", 200, y)
        py5.text(f"{row['メッセージ']}件", 320, y)

    if data:
        active_list = [r['アクティブ'] for r in data]
        avg = sum(active_list) / len(active_list)
        py5.text_size(16)
        py5.text(f"平均: {avg:.1f}人  最大: {max(active_list)}人  最小: {min(active_list)}人", 50, 400)

py5.run_sketch()
サンプル3

棒グラフの描画

CSVデータを棒グラフとして可視化します。これを土台にして、自分なりの可視化を作りましょう。

Python ── sample3_bar.py
import py5
import csv

active_data = []
with open('slack_time.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    count = 0
    for row in reader:
        if count >= 12:
            break
        active_data.append(int(row['アクティブなメンバー数（日次）']))
        count += 1

def setup():
    py5.size(700, 700)
    font = py5.create_font('MS ゴシック', 20)
    py5.text_font(font)

def draw():
    py5.background(255)
    py5.fill(0)
    py5.text_size(18)
    py5.text("アクティブメンバー数の変化", 50, 30)

    for i in range(len(active_data)):
        x = 50 + i * 50
        h = active_data[i]
        py5.fill(100, 150, 255)
        py5.rect(x, 600 - h, 40, h)
        py5.fill(0)
        py5.text_size(12)
        py5.text(str(active_data[i]), x + 15, 590 - h)

py5.run_sketch()

カスタマイズのヒント

  • 色を変える: py5.fill(R, G, B) でバーの色を変更
  • アニメーションを追加: バーが下から伸びるアニメーション
  • マウス操作: マウスオーバーで値をハイライト表示
  • 音を追加: バーをクリックしたときに音を鳴らす
  • 複数系列: メッセージ数とファイル数を並べて比較
