プログラムの3つの基本構造

すべてのプログラムは、以下の3つの構造の組み合わせで作られています。

構造意味Python での例
順次（Sequential）上から順番に1行ずつ実行命令を1行ずつ書く
分岐（Selection）条件によって実行する命令を変えるif / else
反復（Iteration）同じ処理を繰り返すfor / while

今回は順次と反復（for文）を中心に学びます。分岐は第3回で扱います。

料理のレシピと同じです。「材料を切る → 炒める → 盛り付ける」は順次。「味が薄ければ塩を足す」は分岐。「5分間かき混ぜ続ける」は反復です。

Pythonの基本構文（復習）

変数は「名前を付けた箱」。値を入れておき、後から使えます。

# for文の基本形

range(n) は 0 から n-1 までの整数を順番に生成します。

CS基礎: コンピュータの構成要素

コンピュータの5大装置

装置役割日常の例え
CPU（Central Processing Unit）計算・判断を行う「頭脳」料理人の頭脳

アルゴリズムと計算量

アルゴリズムとは

「問題を解くための手順」のこと。繰り返し（ループ）は最も基本的なアルゴリズムです。

計算量の直感

ループの回数が増えると処理時間が増えます。特に二重ループは要注意です。

ループn=10 の処理回数n=100 の処理回数n=1000 の処理回数
1重ループ（O(n)）10回100回1,000回
2重ループ（O(n2)）100回10,000回1,000,000回

step2-5.py の二重ループで体感してみましょう。range の数を変えると描画時間がどう変わるか試してみてください。

1重ループは「教室の全員に1回ずつ握手する」（n回）。2重ループは「全員が全員と握手する」（n×n回）。人数が10人なら100回、100人なら10,000回になります。

データの表現と種類

色 = 数値3つで表現

コンピュータでは、色を3つの数値の組み合わせで表現します。すべての情報は数値化できる ── これがデータサイエンスの出発点です。

  • RGB: 赤(R)・緑(G)・青(B) の光の強さ（各 0〜255）
  • HSB: 色相(H)・彩度(S)・明度(B) で色を指定

データの種類

分類種類例
量的データ
数値で表現		連続データ身長、体重、気温
離散データ人数、回数、個数
質的データ
カテゴリで表現		名義尺度血液型、性別、都道府県
順序尺度成績（A/B/C）、満足度

色のRGB値（0〜255の整数）は離散的な量的データです。

色の変化

ループ変数 i を使って、四角形の色を段階的に変化させます。

Python ── step2-2.py
# step2-2.py - 色の変化
import py5

def setup():
    py5.size(400, 300)
    for i in range(5):
        py5.fill(50 * i, 0, 0)
        py5.rect(50 + i*70, 100, 50, 50)

py5.run_sketch()

考えてみよう

fill(50 * i, 0, 0) で i が 0 のときは fill(0, 0, 0) = 黒、i が 4 のときは fill(200, 0, 0) = 濃い赤になります。

観察してみよう

step2-5

二重ループで格子状に配置

for文を2つ入れ子にすると、格子状（グリッド）に図形を配置できます。

Python ── step2-5.py
# step2-5.py - 二重ループ格子状
import py5

def setup():
    py5.size(400, 400)
    for x in range(5):
        for y in range(5):
            py5.rect(50 + x*70, 50 + y*70, 50, 50)

py5.run_sketch()

二重ループの仕組み

外側のループ（x）が1回進むごとに、内側のループ（y）が5回実行されます。合計 5 × 5 = 25個の四角形が描画されます。

実験: range(5) を range(20) に変えてみよう。描画数は 20×20 = 400個になります。さらに range(50) にすると？

色の表現 RGB・HSB ── 約30分

RGBとHSBの違い

RGB（Red, Green, Blue）

光の三原色。各色 0〜255 で指定します。py5のデフォルトのカラーモードです。

色RGB
赤25500
緑02550
青00255
黄2552550
白255255255
黒000

HSB（Hue, Saturation, Brightness）

人間の感覚に近い色指定方法。「虹のどの色か」「どれくらい鮮やかか」「どれくらい明るいか」で指定します。

パラメータ範囲意味
H（色相）0〜360色の種類（0=赤, 60=黄, 120=緑, 180=シアン, 240=青, 300=紫）
S（彩度）0〜100鮮やかさ（0=灰色, 100=最も鮮やか）
B（明度）0〜100明るさ（0=黒, 100=最も明るい）

HSBの利点: 「虹色のグラデーション」を作りたいとき、Hの値を変えるだけでよい。RGBでは3つの値を同時に調整する必要があり複雑。

実験1

RGB vs HSB の比較

以下のコードを実行して、RGBとHSBで同じ「赤」を表現できることを確認してください。

Python ── experiment1.py
# 実験1 RGB vs HSB comparison
import py5

def setup():
    py5.size(400, 200)
    py5.background(255)
    py5.no_stroke()

    # RGB モード（デフォルト）
    py5.color_mode(py5.RGB, 255)
    py5.fill(255, 0, 0)
    py5.circle(100, 100, 80)

    # HSB モード
    py5.color_mode(py5.HSB, 360, 100, 100)
    py5.fill(0, 100, 100)
    py5.circle(300, 100, 80)

py5.run_sketch()

確認ポイント

  • 左の円（RGB: 255, 0, 0）と右の円（HSB: 0, 100, 100）は同じ赤色になる
  • color_mode でカラーモードを切り替えられる
  • HSBの H=0 は赤（色相環の開始位置）

発展

マウスで色を操作する（参考例）

draw() 関数を使うとアニメーションが作れます。マウスの位置で色が変わるインタラクティブな例を試してみましょう。

Python ── interactive_hsb.py
import py5

def setup():
    py5.size(800, 800)
    py5.color_mode(py5.HSB, 360, 100, 100)
    py5.no_stroke()

def draw():
    py5.background(0)
    hue = py5.remap(py5.mouse_x, 0, py5.width, 0, 360)
    brightness = py5.remap(py5.mouse_y, 0, py5.height, 100, 0)
    py5.fill(hue, 100, brightness)
    py5.ellipse(py5.width / 2, py5.height / 2, 200, 200)

py5.run_sketch()

setup() と draw() の違い

関数実行回数用途
setup()1回だけ初期設定（ウィンドウサイズ、背景色など）
draw()毎フレーム繰り返し（約60fps）アニメーション、ユーザー操作への反応

remap() は値の範囲を変換する関数です。マウスのx座標（0〜800）を色相（0〜360）に変換しています。
