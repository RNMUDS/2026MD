時系列データ・加速度センサ・Jupyter Notebook

ノート

本日の流れ

  1. 時系列データとは: 時間の経過に沿って記録されたデータ（センサ値、株価、気温など）
  2. 加速度センサの仕組み: 3軸（X/Y/Z）で重力・動きを検出、スマートフォンにも搭載
  3. CSVとJSON: 表形式データの2つの代表的フォーマット
  4. Jupyter Notebook: コードと実行結果を1つのドキュメントにまとめる対話型環境

時系列データ（Time Series Data）

時間の経過とともに記録されたデータを時系列データと呼びます。一定間隔でデータを取得することをサンプリング、その頻度をサンプリングレート（例: 100Hz = 1秒間に100回取得）と呼びます。

体温計で毎朝体温を測り記録するのと同じです。1日1回なら「サンプリングレート = 1回/日」です。加速度センサは1秒間に数十〜数百回測定します。

加速度センサ（Accelerometer）

物体にかかる加速度を3軸（X: 左右、Y: 上下、Z: 前後）で測定するセンサです。静止状態でも重力加速度（約9.81 m/s²）がZ軸方向に検出されます。

軸方向スマホを水平に持ったとき
X左右左右に傾けると変化
Y上下前後に傾けると変化
Z前後（垂直）静止時 ≈ 9.81 m/s²

欠損値と外れ値

  • 欠損値（Missing Value）: データが記録されなかった箇所。CSVでは空のセル（0.03,）として現れる
  • 外れ値（Outlier）: 他のデータと比べて極端に大きい・小さい値。センサの誤作動やノイズが原因のことが多い

ログデータ

システムやアプリケーションの動作を時系列で記録したデータをログ（Log）と呼びます。センサデータもログデータの一種です。タイムスタンプ付きで記録されるため、後から「いつ何が起きたか」を分析できます。

Step 3: ベースコード ─ CSVデータの表示

標準課題

以下のコードを step5.ipynb のセルに入力して実行してください。CSVファイルの先頭5行が表示されます。

Python ── ベースコード
import py5

lines = []

def setup():
    py5.size(400, 400)
    global lines
    lines = py5.load_strings("acc_y.csv")

def draw():
    global lines
    py5.background(255)
    py5.fill(0)
    py5.text_size(18)
    margin = 0
    line_height = 18
    for i in range(5):
        line = lines[i].strip()
        columns = line.split(',')
        text = f"{columns[0]} : {columns[1]}"
        py5.text(text, margin, margin + i * line_height)

py5.run_sketch()

上のベースコードを step5.ipynb にコピーして実行してみましょう。表示結果のスクリーンショットをNotionに記録してください。

問題点に気づきましたか？

このコードには2つの問題があります。実行して確認しましょう:

  • 問題1: margin = 0 のため、テキストが画面の左上に張り付いて見切れている
  • 問題2: 5行しか表示されず、ヘッダー行も含まれている

修正課題1: margin を修正する

margin と line_height の値を調整して、テキストが画面内に収まるようにしてください。表示行数も増やしてみましょう。

修正課題2: AccY列のみを表示する

ヘッダー行（1行目）をスキップし、AccY列（columns[1]）の値だけを表示するように修正してください。

ヒント: margin の修正

margin = 20 のように余白を設定し、line_height = 22 に増やすと読みやすくなります。

range(5) を range(len(lines)) や range(min(20, len(lines))) に変えると表示行数を増やせます。

ヒント: ヘッダースキップ

ヘッダーをスキップするには、ループの開始を range(1, ...) にします。AccYだけを表示するには columns[1] のみを使います。

Python ── ヒント
for i in range(1, min(20, len(lines))):
    line = lines[i].strip()
    columns = line.split(',')
    y_value = columns[1]
    py5.text(y_value, margin, margin + (i - 1) * line_height)

std5-2: グループワーク2 ─ 時系列データアニメーション（約50分）

Step 1: CSVデータでアニメーション

ノート

CSVから読み込んだAccY値を使って、円を動かすアニメーションを作ります。

Python ── アニメーション基本コード
import py5

y_values = []
frame_index = 0

def setup():
    py5.size(400, 400)
    lines = py5.load_strings("acc_y.csv")
    for line in lines[1:]:
        parts = line.split(',')
        y = float(parts[1])
        y_values.append(y)

def draw():
    global frame_index
    py5.background(255)
    if frame_index < len(y_values):
        y = y_values[frame_index]
        py5.fill(0, 0, 255)
        py5.ellipse(200, y, 30, 30)
        frame_index += 1
    else:
        py5.no_loop()

py5.run_sketch()

円が画面外に飛び出す？

加速度の値は -10〜10 程度の小さな値のため、画面サイズ（400px）に対してほとんど動きません。正規化（remap）が必要です。

Step 2: py5.remap() で値を画面に合わせる

ノート

正規化・remap とは

ある範囲の値を別の範囲に変換することを正規化（remap）と呼びます。py5.remap() を使うと、加速度の値（例: -10〜10）を画面のY座標（0〜400）に変換できます。

温度計の目盛りと同じです。摂氏0〜100度を、温度計の0〜30cmの液柱に「変換」して表示しています。 
Python ── remap の使い方
# py5.remap(値, 元の最小, 元の最大, 変換先の最小, 変換先の最大)
screen_y = py5.remap(y, -10, 10, 0, 400)

# 例: y = 5 の場合
# remap(5, -10, 10, 0, 400) → 300

draw() 内の y を py5.remap() で変換してみましょう:

Python ── remap を適用した draw()
def draw():
    global frame_index
    py5.background(255)
    if frame_index < len(y_values):
        y = y_values[frame_index]
        screen_y = py5.remap(y, -10, 10, 0, 400)
        py5.fill(0, 0, 255)
        py5.ellipse(200, screen_y, 30, 30)
        frame_index += 1
    else:
        py5.no_loop()

Step 3: 時間軸を加える

ノート

円のX座標を frame_index に連動させると、時間の経過に沿ってデータが流れるアニメーションになります。

Python ── 時間軸の追加
# X座標も frame_index から計算
screen_x = py5.remap(frame_index, 0, len(y_values), 0, 400)
py5.ellipse(screen_x, screen_y, 10, 10)

軌跡を残すには？

py5.background(255) を draw() の先頭から削除するか、py5.background(255, 20)（半透明の白）に変えると、これまでの軌跡が薄く残ります。

remap の範囲を変えて円の動きがどう変わるか試しましょう。また background(255, 20) に変えて軌跡表示も試してみてください。変更前後をNotionに記録してください。

Step 4: UFO画像で表示する

標準課題

青い円の代わりにUFO画像を表示しましょう。画像ファイル（ufo.png）をGoogle Driveからダウンロードし、~/MD1/no5/ に配置してください。

Python ── UFO画像を使ったアニメーション
import py5

y_values = []
frame_index = 0
ufo_img = None

def setup():
    global ufo_img
    py5.size(400, 400)
    ufo_img = py5.load_image("ufo.png")
    lines = py5.load_strings("acc_y.csv")
    for line in lines[1:]:
        parts = line.split(',')
        y = float(parts[1])
        y_values.append(y)

def draw():
    global frame_index
    py5.background(255)
    if frame_index < len(y_values):
        y = y_values[frame_index]
        screen_y = py5.remap(y, -10, 10, 0, 400)
        py5.image(ufo_img, 200 - 25, screen_y - 25, 50, 50)
        frame_index += 1
    else:
        py5.no_loop()

py5.run_sketch()

py5.image() の引数

py5.image(画像, x, y, 幅, 高さ) で画像を描画します。x, y は画像の左上の座標なので、中央に配置するために画像サイズの半分を引いています。
