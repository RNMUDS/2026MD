CS/DS基礎: データ構造と関数

データ構造とは

プログラムで扱うデータの整理方法のことです。目的に応じて適切なデータ構造を選ぶことで、コードがシンプルで効率的になります。

データ構造特徴日常の例Python
リスト順番のあるデータの集合買い物リスト、出席番号[1, 2, 3]
辞書キーと値のペア英和辞典、電話帳{"名前": "太郎"}

関数とは

処理のまとまりに名前を付けたものです。引数（材料）を受け取り、処理（調理）をして、戻り値（料理）を返します。

引数（入力） → 関数（処理） → 戻り値（出力）

なぜ関数に分割するか

  • 再利用: 同じ処理を何度も書かなくてよい
  • 可読性: コードに名前がつくので読みやすい
  • テスト容易性: 小さい単位でテストしやすい
  • 修正容易性: 1か所直せば全体に反映される

リストの基本 ── グループ演習・約25分

リスト（list）とは

複数のデータを順番に並べて1つの変数で管理できるデータ構造です。角カッコ [ ] で囲み、カンマ , で区切ります。

リストは「ロッカーの列」です。0番、1番、2番...と番号（インデックス）がついていて、番号を指定して中身を取り出せます。
ノート

リストの基本操作

1. リストの作成

Python
# リストの作成
fruits = ["りんご", "バナナ", "みかん"]
numbers = [10, 20, 30, 40, 50]
colors = [(255, 0, 0), (0, 255, 0), (0, 0, 255)]  # タプルのリスト

2. 要素の取得（インデックス）

Python
fruits = ["りんご", "バナナ", "みかん"]
print(fruits[0])   # りんご（最初の要素は0番目）
print(fruits[1])   # バナナ
print(fruits[-1])  # みかん（末尾から1番目）

インデックスは0から始まる

Pythonでは最初の要素は [0] です。[1] ではありません。存在しないインデックスを指定すると IndexError が発生します。

3. 要素の追加・長さ

Python
fruits = ["りんご", "バナナ"]
fruits.append("みかん")   # 末尾に追加
print(fruits)             # ['りんご', 'バナナ', 'みかん']
print(len(fruits))        # 3（要素の数）

4. スライス

Python
numbers = [10, 20, 30, 40, 50]
print(numbers[1:3])   # [20, 30]（1番目から3番目の手前まで）
print(numbers[:3])    # [10, 20, 30]（最初から3番目の手前まで）
print(numbers[2:])    # [30, 40, 50]（2番目から最後まで）

5. for文との組み合わせ

Python
fruits = ["りんご", "バナナ", "みかん"]
for fruit in fruits:
    print(fruit)
# りんご
# バナナ
# みかん
操作コード説明
作成[1, 2, 3]角カッコで囲む
取得list[0]インデックスで取得
追加list.append(x)末尾に追加
長さlen(list)要素の数を返す
スライスlist[1:3]範囲で取り出す
繰り返しfor x in list:1つずつ取り出す

py5でリストを使う: 座標リストから複数の円を描画

リストに色情報（RGBタプル）を入れておき、for文で順番に描画します。

Python ── list_circles.py
# 5色の円をリストで描画する
colors = [(255, 0, 0), (0, 255, 0), (0, 0, 255), (255, 255, 0), (255, 0, 255)]

import py5

def setup():
    py5.size(500, 300)
    py5.background(30)
    py5.no_stroke()
    for i in range(len(colors)):
        r, g, b = colors[i]
        py5.fill(r, g, b)
        py5.circle(50 + i * 100, 150, 60)

py5.run_sketch()

ポイント: range(len(list))

辞書の基本 ── グループ演習・約25分

辞書（dict）とは

キー（名前）と値（データ）のペアでデータを管理するデータ構造です。波カッコ { } で囲み、キー: 値 の形式で書きます。

辞書は「電話帳」です。「名前（キー）」を指定すると「電話番号（値）」がすぐに見つかります。リストとの違いは、番号（0, 1, 2...）ではなく名前で検索できることです。
ノート

辞書の基本操作

1. 辞書の作成と値の取得

Python
# 辞書の作成
student = {
    "name": "太郎",
    "age": 20,
    "grade": "B2",
}

# 値の取得
print(student["name"])   # 太郎
print(student["age"])    # 20

2. 値の追加・変更

Python
student["email"] = "taro@example.com"  # 新しいキーで追加
student["age"] = 21                      # 既存のキーで上書き
print(student)

3. keys() / values() / items()

Python
student = {"name": "太郎", "age": 20}

print(student.keys())    # dict_keys(['name', 'age'])
print(student.values())  # dict_values(['太郎', 20])

for key, value in student.items():
    print(f"{key}: {value}")
# name: 太郎
# age: 20

4. キーの存在確認（in）

Python
if "name" in student:
    print("名前が登録されています")

if "phone" not in student:
    print("電話番号は未登録です")

py5で辞書を使う: キー入力で色を変える

color_dict = {
    'r': (255, 0, 0),
    'g': (0, 255, 0),

自分で関数を作る

これまで py5.circle() や len() など「誰かが作った関数」を使ってきました。今回は自分で関数を定義して、処理を整理します。

関数は「自動販売機」です。お金（引数）を入れてボタンを押すと、ジュース（戻り値）が出てきます。中の仕組み（処理）を知らなくても使えます。
ノート

関数の基本構文

1. 引数なし・戻り値なしの関数

Python
def greet():
    print("こんにちは！")

greet()  # こんにちは！
greet()  # こんにちは！（何度でも呼べる）

2. 引数ありの関数

Python
def greet(name):
    print(f"こんにちは、{name}さん！")

greet("太郎")   # こんにちは、太郎さん！
greet("花子")   # こんにちは、花子さん！

3. 戻り値ありの関数（return）

Python
def add(a, b):

result = add(3, 5)

def greet(name, greeting="こんにちは"):

greet("花子", "おはよう")  # おはよう、花子さん！

py5で関数を使う: draw_house() で家を量産

家を描く処理を関数にまとめると、座標と色を変えるだけで何軒でも描けます。

Python ── draw_houses.py
# 関数で家を量産する
import py5

def draw_house(x, y, wall_color, roof_color):
    # 壁（四角形）
    py5.fill(*wall_color)
    py5.rect(x, y, 80, 60)
    # 屋根（三角形）
    py5.fill(*roof_color)
    py5.triangle(x - 10, y, x + 40, y - 40, x + 90, y)

def setup():
    py5.size(600, 300)
    py5.background(135, 206, 235)  # 空色
    draw_house(50, 200, (139, 69, 19), (255, 0, 0))
    draw_house(200, 200, (160, 82, 45), (0, 100, 0))
    draw_house(350, 200, (210, 180, 140), (0, 0, 200))

py5.run_sketch()

関数化のメリット

もし関数を使わなかったら、壁と屋根の描画コードを3回コピー＆ペーストする必要があります。関数化すると:

  • コードが短くなる（DRY原則: Don't Repeat Yourself）
  • 壁の幅を変えたいとき、1か所の修正で全ての家に反映される
  • 「この関数は何をするか」が名前から分かる

応用: リスト + 辞書 + 関数の組み合わせ

辞書のリストを関数で処理すると、データ管理が整理されます。次回（第5回）のCSVデータ処理はこのパターンの延長です。

Python ── combo_example.py
# リスト + 辞書 + 関数の組み合わせ
import py5

# 辞書のリスト（データ）
circles_data = [
    {"x": 100, "y": 150, "size": 60, "color": (255, 0, 0)},
    {"x": 250, "y": 150, "size": 80, "color": (0, 255, 0)},
    {"x": 400, "y": 150, "size": 50, "color": (0, 0, 255)},
]

def draw_circle(data):
    """辞書のデータから円を描画する"""
    py5.fill(*data["color"])
    py5.circle(data["x"], data["y"], data["size"])

def setup():
    py5.size(500, 300)
    py5.background(30)
    py5.no_stroke()
    for c in circles_data:
        draw_circle(c)

py5.run_sketch()

  1. 自分でオリジナルの描画関数を1つ以上定義する（例: draw_tree(x, y, color)、draw_star(x, y, size) など）
  2. 関数は引数を2つ以上受け取るようにする
  3. 関数を3回以上呼び出して、異なる引数で描画する
  4. for文やリストと組み合わせても良い

アイデア例

  • draw_tree(x, y, trunk_color, leaf_color) ── 木を描く関数で森を作る
  • draw_flower(x, y, petal_color, size) ── 花を描く関数で花畑を作る
  • draw_face(x, y, eye_color) ── 顔を描く関数で集合写真を作る
